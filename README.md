# Elective-Mini-Project-1
This is the repository for my elective mini project 1, where I will be creating a game (tic tac toe)


# What I Built
I built a fully playable Tic-Tac-Toe game that runs in the browser using HTML, CSS, and JavaScript. The game allows two players to take turns, keeps track of scores, and provides visual feedback for wins.


# How to Play
- Player X starts first
- Click on any empty cell to place your mark (X or O)
- Players alternate turns
- The first player to get 3 in a row (horizontal, vertical, or diagonal) wins
- The winning combination is highlighted with a green background and a red line
- Click the **Restart Game** button to play again without refreshing
